## Install

```bash
  $ npm i
```

## Start local project

```bash
  $ cp .env.example .env
  $ sls offline start
```

## Deploy

```bash
  $ sls deploy -s dev // code
  $ cd database && sls deploy -s dev // db
```

## NVM

```bash
  $ nvm install v15.5.0
  $ nvm alias default v15.5.0
  $ nvm use v15.5.0
```
